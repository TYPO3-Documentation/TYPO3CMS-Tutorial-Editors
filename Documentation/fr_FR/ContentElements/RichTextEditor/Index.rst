L'éditeur de texte enrichi
^^^^^^^^^^^^^^^^^^^^^^^^^^

L'éditeur va certainement se sembler familier car il n'est pas
très différent d'un traitement de texte. Les options disponibles
dans les barres d'outils varient en fonction de la façon dont
l'installation a été mis en place.

.. figure:: ../../Images/EditContentRichTextEditor.png
   :alt: Une vue typique de l'éditeur de texte enrichi


Une caractéristique particulière est la création de lien.
Une fois que vous avez sélectionné un texte,
vous pouvez cliquer sur l'icône indiquée ci-dessus pour créer un lien.


.. _rte-linking:

Création d'un lien
""""""""""""""""""

Cliquez sur l'icône "Insérer un lien" pour ouvrir une boîte de dialogue
vous permettant de sélectionner ce vers quoi vous voulez faire un lien.
Ceci s'appele le "navigateur de lien".

.. figure:: ../../Images/LinkBrowser.png
   :alt: Le navigateur de lien de TYPO3 CMS


Le navigateur de lien vous permet de faire des liens vers des pages,
des fichiers et dossiers ou une URL externe ou une adresse électronique.
L'onglet "Page" (1) est actif par défaut.

Développez l'arborescence de page jusqu'à ce que vous voyiez
la page "A propos > TYPO3" (2), puis cliquez sur la petite flèche
à droite du titre de la page (3). Cela déclenche l'affichage
- dans la moitié droite de la fenêtre - des éléments de contenu
situés sur cette page.

Cliquez sur l'élément "TYPO3 - Le CMS d'Enterprise" (4) pour créer
un lien vers cet élément. Cela va automatiquement créer un lien comprenant
une ancre pointant vers cet élément de contenu (TYPO3 CMS place automatiquement
des ancres sur chaque élément de contenu lors du rendu).

Vous auriez pu également avoir cliqué sur la page "TYPO3" (2) pour créer
un lien vers cette page sans cibler un élément de contenu spécifique.

Enregistrez et visualisez votre travail dans le frontend.
Il devrait y avoir un lien hypertexte vers la page cible sélectionnée.
