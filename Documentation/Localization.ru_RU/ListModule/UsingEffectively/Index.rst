========
Введение
========

Модуль :guilabel:`Веб > Список` / :guilabel:`Web > List` позволяет перемещаться по страницам или папкам сайта и просматривать все записи, хранящиеся на них. Модуль :guilabel:`List` также позволяет создавать и управлять записями, для которых нет специального модуля.

Примерами различных типов записей, которые могут храниться на данной странице, являются подстраницы, элементы содержимого и новостные сообщения.

==================================
Использование модуля Список / List
==================================

Выберите модуль :guilabel:`Веб > Список` / :guilabel:`Web > List` и перейдите на страницу "Congratulations". Вы должны увидеть следующее:

.. include:: /Images/AutomaticScreenshots/ListModule/ListModule.rst.txt

В режиме списка отображается список записей, хранящихся на текущей странице, сгруппированных по типу. Название, которое отображается для каждой записи, зависит от того, какое поле используется в качестве метки. Для "pages" (страниц) этим полем является "Pagetitle".

В режиме списка в верхней части окна предлагается несколько пиктограмм различных действий:

1. :guilabel:`Создать` / :guilabel:`Create` новую запись (появится мастер, позволяющий выбрать тип записи).

2. :guilabel:`Просмотр` / :guilabel:`View` текущей страницы (на сайте).

3. :guilabel:`Правка` / :guilabel:`Edit` текущей страницы.

4. :guilabel:`Поиск` / :guilabel:`Search` на текущей странице.

5. :guilabel:`Очистить кэш` / :guilabel:`Clear the cache` текущей страницы.

6. :guilabel:`Обновить` / :guilabel:`Refresh` представление списка.

7. :guilabel:`Закладка` / :guilabel:`Bookmark` для этого раздела.

8. :guilabel:`Копировать URL` / :guilabel:`Copy URL` для этого раздела.

9. Доступ к :guilabel:`контекстной справке` (удалено в версии 12.0).

В **Области содержимого** также доступны значки действий:

.. include:: /Images/AutomaticScreenshots/ListModule/ListModuleContentArea.rst.txt


#. Создать :guilabel:`новую запись на странице`.

#. Значки действий для данной записи. Вид значков зависит от выбранного типа записи. Если запись доступна для просмотра на сайте (например, "страницы" / "pages"), то отображается значок "Просмотр". Все записи будут иметь следующие три пиктограммы, соответственно "Редактировать", "Скрыть/Убрать" (т.е. видна или нет на сайте) и "Удалить".

   Значок многоточия (три последовательные точки) позволяет развернуть список значков и показать дополнительные значки действий (как правило, для доступа к информации, перемещения и т. д.).

   Наконец, значки вырезать и вставить.

.. todo: the "Extended view" option does not exist anymore

Если вы хотите, чтобы список значков был всегда развёрнут, можно установить флажок :guilabel:`Расширенный вид` / :guilabel:`Extended view` в самом низу диалогового окна.

Также убедитесь, что установлен флажок :guilabel:`Показывать буфер обмена` / :guilabel:`Show clipboard`, поскольку далее речь пойдет именно об этом.

Если вы планируете управлять многоязычным веб-сайтом, то вам необходимо установить флажок :guilabel:`Localization view` / :guilabel:`Режим локализации`.
