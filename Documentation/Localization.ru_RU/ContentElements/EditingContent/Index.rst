.. _content-editing:

==========================
Редактирование содержимого
==========================

..  youtube:: JR3sOeRImTo

------------

На странице щелкните значок карандаша у элемента содержимого, который необходимо отредактировать, затем отредактируйте текст и внесите другие изменения по мере необходимости.

Также можно изменить тип элемента содержимого, выбрав новый элемент из списка :guilabel:`Тип` / :guilabel:`Type`.

.. include:: /Images/AutomaticScreenshots/EditContent/EditContentChangeType.rst.txt

При изменении типа элемента содержимого система TYPO3 CMS перезагружает экран для отображения актуальные поля свойств для элемента другого типа.

.. include:: /Images/AutomaticScreenshots/EditContent/EditContentChangeTypeConfirm.rst.txt

Это действие не удаляет уже введенные свойства содержимого. Оно сохраняется в фоновом режиме и будет восстановлено при повторном выборе соответствующего (предыдущего) типа элемента содержимого.
