========
Страницы
========

.. container:: row m-0 p-0

   .. container:: col-md-6 pl-0 pr-3 py-3 m-0

      .. container:: card px-0 h-100

         .. rst-class:: card-header h3

            .. rubric:: :ref:`Создание страниц<pages-creating>`

         .. container:: card-body

            Создание страниц - это одно из первых действий, которые вы будете выполнять после установки новой версии TYPO3. В этой главе содержится информация о создании страниц и их размещении в дереве страниц.


   .. container:: col-md-6 pl-0 pr-3 py-3 m-0

      .. container:: card px-0 h-100

         .. rst-class:: card-header h3

            .. rubric:: :ref:`Создание нескольких страниц<pages-multiple>`

         .. container:: card-body

            В TYPO3 имеется встроенная функция, для одновременного создания нескольких страниц, что облегчает начальную работу с большими проектами. В этой главе представлена информация об использовании этого инструмента.

   .. container:: col-md-6 pl-0 pr-3 py-3 m-0

      .. container:: card px-0 h-100

         .. rst-class:: card-header h3

            .. rubric:: :ref:`Работа со страницами<pages-working-with>`

         .. container:: card-body

            В этой главе рассказывается о том, как можно перемещать, копировать и удалять страницы в дереве страниц, а также показано, как использовать инструмент фильтрации.

   .. container:: col-md-6 pl-0 pr-3 py-3 m-0

      .. container:: card px-0 h-100

         .. rst-class:: card-header h3

            .. rubric:: :ref:`Типы страниц<pages-types>`

         .. container:: card-body

            В дереве страниц могут быть созданы страницы различных типов, включая страницы с ограниченным доступом и ссылки на другие страницы. В этой главе приведена информация о различных типах страниц и способах их использования.


   .. container:: col-md-6 pl-0 pr-3 py-3 m-0

      .. container:: card px-0 h-100

         .. rst-class:: card-header h3

            .. rubric:: :ref:`Свойства страниц<pages-properties>`

         .. container:: card-body

            Каждая страница в дереве страниц имеет набор изменяемых свойств. В частности, это касается используемого шаблона, SEO-информации и видимости страницы на сайте. В этой главе подробно рассматривается каждая из вкладок свойств и способы их настройки.
