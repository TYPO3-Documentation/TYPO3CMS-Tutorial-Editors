Типы страниц, используемые по умолчанию
=======================================

По умолчанию в TYPO3 предусмотрены следующие типы страниц.

Страница
--------

Стандартная
    Это тип страницы по умолчанию и самый распространенный. Он удовлетворяет всем основным требованиям.

Раздел внутренних пользователей
    Этот тип страницы отображается во внешнем интерфейсе только для определенной группы пользователей внутреннего интерфейса. Для просмотра этого типа страниц необходимо войти в систему.

Ссылка
------

Ссылка
    Этот тип страницы является ссылкой на другую страницу в дереве страниц. Когда пользователи переходят на эту страницу на сайте, они сразу же попадают на целевую страницу.

Точка монтирования
    Точка монтирования позволяет выбрать любую другую страницу в дереве страниц. Все дочерние страницы выбранной страницы будут отображаться как дочерние страницы точки монтирования. Это позволяет дублировать части дерева страниц с точки зрения навигации, не дублируя страницы и содержимое.

    Дополнительную информацию о точках монтирования смотрите в разделе :ref:`Mounts <t3coreapi:access-options-mounts>` руководства "TYPO3 Explained" / "TYPO3 в деталях".

Ссылка на внешний URL
    Этот тип страницы аналогичен типу :guilabel:`Ссылка`, но ведет пользователя на страницу другого сайта.

Специальный
-----------

Папка
    Тип страницы папки представляет собой контейнер. Обычно он используется для хранения записей, отличных от страниц или элементов содержимого. Он не отображается на сайте.

Корзина
    ..  versionchanged:: 13.0
        Тип Корзина / recycler был удален. В качестве замены используйте модуль :doc:`корзина<ext_recycler:Index>`.

Разделитель для меню
    Этот тип страницы создает визуальное разделение в дереве страниц. С помощью TypoScript можно отобразить эти разделители в элементах навигации на сайте.


Пользовательские типы страниц
=============================

В зависимости от проекта могут быть доступны и пользовательские типы страниц.
