===========================
Создание нескольких страниц
===========================

#. В дереве страниц установите курсор на страницу, которая должна стать родительской для вновь создаваемых страниц.

#. Щёлкните правой кнопкой мыши на странице и выберите в контекстном меню :guilabel:`Дополнительные настройки... > Создать несколько страниц` / :guilabel:`More options > Create multiple pages`.

   .. figure:: ../../Images/ManualScreenshots/PageTree/MultiplePagesContextMenu.png
      :alt: В контекстном меню выберите пункт "Создать несколько страниц".
      :class: with-border

   Появится диалог :guilabel:`Создание нескольких страниц` / :guilabel:`Create Multiple Pages` с пятью новыми пустыми страницами.

   .. figure:: ../../Images/ManualScreenshots/Page/MultiplePagesCreatePages.png
      :alt: Диалог "Создание нескольких страниц"
      :class: with-border

#. Задайте каждой новой странице название и выберите ее тип.

#. Нажмите кнопку :guilabel:`Добавить строки` / :guilabel:`Add more lines`, для добавления еще пяти новых страниц, если требуется создать больше страниц.

#. Настройте новые страницы с помощью опций в нижней части экрана:

   *  Опция :guilabel:`Размещать новые страницы после существующих подстраниц` / :guilabel:`Place new pages after the existing subpages` добавляет новые страницы под существующими подстраницами. В противном случае они будут размещены над текущими страницами.

   *  Опции :guilabel:`Скрыть новые страницы` / :guilabel:`Hide new pages` и :guilabel:`Скрыть новые страницы в меню` / :guilabel:`Hide new pages in menus` позволяют управлять видимостью новых страниц.

      К скрытой странице никто не может получить доступ через веб-сайт. Страница, скрытая в меню, не отображается в навигации сайта, но доступ к ней может получить тот, кто имеет прямой URL этой страницы.

#. Для создания страниц нажмите кнопку :guilabel:`Создать страницы` / :guilabel:`Create pages`.
