=====
Файлы
=====

Модуль :guilabel:`Файл > Список файлов` / :guilabel:`File > Filelist` - это место, где можно управлять всеми медиафайлами, связанными с сайтом TYPO3.

.. index::
   Files; Management
   Directories; fileadmin
.. _managing-files:
.. _managing-files-in-typo:

Управление файлами в CMS TYPO3
==============================

Управление файлами, включая документы и изображения, осуществляется в модуле Список файлов / Filelist. Подобно модулю :guilabel:`Веб > Список` :guilabel:`Web > List`, здесь отображается дерево навигации, соответствующее файловой структуре на сервере, и список всех файлов для выбранного каталога.

.. include:: /Images/AutomaticScreenshots/FilelistModule/FilelistModule.rst.txt

Для пользователей-администраторов отображаемая папка по умолчанию называется "fileadmin/ (auto-generated)" и соответствует папке :file:`fileadmin/`, расположенной в корневой папке веб-сервера.

Использование этих файлов внутри элементов содержимого для их отображения или ссылок на них на сайте рассматривается в главе :ref:`изображения<images>`.

.. note::

   Существуют расширения, позволяющие подключаться к удаленным пулам хранения (например, серверу WebDAV или учетной записи Amazon S3) и работать с файлами так, как если бы они находились на сервере TYPO3 CMS.

Метаданные файла
----------------

Можно указать метаданные для файла, нажав на кнопку, выглядящую как карандаш с надписью :guilabel:`Править метаданные этого файла` / :guilabel:`Edit Metadata of this file`.

Предоставляемые метаданные могут содержать (в зависимости от настроек системы и формата файла):

*  Текст, который будет отображаться для всех пользователей в некоторых контекстах (:guilabel:`Название` / :guilabel:`Title`, :guilabel:`Описание` / :guilabel:`Description`).
*  Текст для :ref:`доступности`, например, :guilabel:`альтернативный текст - alt` для изображения или :guilabel:`название при загрузке` / :guilabel:`download name`.
*  Данные, обязательные в связи с вопросами авторского права.
*  Метаданные, прикрепленные к загруженному файлу, например, информация об использованной камере или месте съемки.
